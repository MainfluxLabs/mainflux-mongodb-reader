Mainflux HTTP API for message reading from MongoDB.

### Installation
Use [`go`](https://golang.org/cmd/go/) tool to "get" (i.e. fetch and build) `mainflux-mongodb-reader` package:
```bash
go get github.com/mainflux/mainflux-mongodb-reader
```
